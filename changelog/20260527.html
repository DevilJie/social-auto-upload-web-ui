一键启动脚本

30+ 次提交
Linux/macOS 启动脚本 新功能

start.sh — Bash 脚本，支持 Linux 和 macOS

  • 运行时环境检查（Python、Node、npm、curl、ffmpeg、ffprobe）
  • Python venv 自动创建和依赖管理
  • 端口冲突自动处理（kill 占用进程）
  • 服务健康检查（自动检测后端实际端口）
  • 后端日志实时显示（tail -f）
Windows 启动脚本 新功能

start.bat — CMD 批处理脚本

  • 纯 CMD 实现，不依赖 PowerShell
  • 端口检测使用 netstat
  • 进程 PID 管理（精准停止服务）
  • 依赖安装进度显示
  • 新增 stop.bat 用于停止服务
跨平台兼容 重构

统一的 6 步启动流程

  • 检查运行时环境
  • 处理端口冲突
  • 准备后端环境（venv + CloakBrowser）
  • 准备前端环境（npm）
  • 启动服务
  • 等待服务就绪并显示入口
