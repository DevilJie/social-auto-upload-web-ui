04 / COMPOSABLE

useBatchSetApply 组合函数

视频/图集各自的 composable,基于 useChannelForm 公共 API 实现 — 弹窗确认后批量写入平台/账号配置。

  • useBatchSetApply (视频) 接收 BatchSetDialog 抛出的 form 数据,遍历勾选的 accountId,调用 useChannelForm.setAccountOverride 写入;apply 后强制刷新当前 form。
  • useImageBatchSetApply (图集) 图集专属 composable,适配 ImagePublish 的 panel 结构,修复 panel 方法查找路径问题。
  • setPlatformConfig 为整个平台(所有账号)统一写入配置,如声明、分类等;支持增量合并,不会清空已有字段。
  • getCheckedAccountIds 读取当前弹窗中勾选的所有 accountId,作为写入范围;与 ChannelSummary / PublishChannelList 联动。

API 路径

  • setPlatformConfig(platform, cfg)
  • setAccountOverride(id, cfg)
  • getCheckedAccountIds()
  • 强制刷新当前 form

06 / SCRAPER & ANTI-HOTLINK

微博抓取器 + 防盗链代理

专用 scrape_weibo_profile 抓取昵称/头像,头像走后端代理绕过 sinaimg.cn 防盗链 403,JS eval 兜底兼容。

  • JS eval 兜底 scrape_weibo_profile 改用 JS eval 提取头像/昵称,避免 Playwright locator API 在微博 SPA 上的兼容问题。
  • 头像走后端代理 头像请求走后端代理,绕过 sinaimg.cn 防盗链 403;前端 img 加 referrerpolicy="no-referrer"。
  • scraper 范围限定 登录检测 + scraper 限定到 .woo-tab-nav 顶部导航,避免误抓页面其他位置的相似元素。
