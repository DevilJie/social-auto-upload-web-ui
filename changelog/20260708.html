Cookie 一键导入 · 15 平台账号打通

v1.2.0 是一次账号管理升级版本 — 新增Cookie 字符串一键导入账号能力,全部 15 个平台(抖音/B站/百家号/今日头条/知乎/微博/视频号/小红书/快手/TikTok/YouTube/腾讯视频/爱奇艺/支付宝/CSDN)接入同一套逻辑,无需扫码登录即可绑定已有账号。账号管理界面重设计:导入用户左右分栏弹窗(左侧平台扁平卡片 + 搜索 + 滚动,右侧 Cookie 输入),4 步进度条实时反馈解析/生成/同步/完成状态,4 个启动脚本启动时强制更新到最新版本。

15 平台支持 25 文件变更 5 个提交 4 大主题

核心亮点

四大方向,账号管理体验大幅提升

Cookie 一键导入账号

所有 15 个平台支持从浏览器粘贴 cookie 字符串直接导入账号,无需扫码登录。新增 BasePlatform.import_cookie 通用 4 步流程,子类只需加 3 行 (supports_cookie_import + domain + parse hook) 即可接入。

核心特性

导入弹窗 UI 重设计

左右分栏布局:左侧平台扁平卡片列表(搜索过滤 + 垂直滚动 + 选中态品牌色边框),右侧 cookie 输入(等宽字体 + 提示框)。4 步 el-steps 进度展示(解析/生成/同步/完成) + 完成态账号预览卡片 + SSE 实时反馈。

体验

启动脚本强制更新

start.sh / start-beta.sh / start.bat / start-beta.bat 启动时直接 git fetch + reset --hard 拉取最新版本,去除"是否更新 Y/n"交互。本地任何未提交修改会被覆盖,确保所有人始终运行最新代码。

详细功能

每个亮点的具体改动与影响

✓

BasePlatform.import_cookie 抽象 + 通用流程

BasePlatform 加 2 个类属性 (supports_cookie_import / platform_cookie_domain) + 1 个抽象 hook (_parse_cookie_to_storage_state)。基类 import_cookie 封装 4 步流程(解析/生成/sync/写库),子平台只需 3 行接入。cookie 失效时自动清理临时文件,不留垃圾账号。

✓

15 平台 cookie 域映射

抖音 .douyin.com / B站 .bilibili.com / 百家号 .baidu.com / 今日头条 .toutiao.com / 知乎 .zhihu.com / 微博 .weibo.com / 视频号 .qq.com / 小红书 .xiaohongshu.com / 快手 .kuaishou.com / TikTok .tiktok.com / YouTube .youtube.com / 腾讯视频 .qq.com / 爱奇艺 .iqiyi.com / 支付宝 .alipay.com / CSDN .csdn.net。复制 3 行就能接入任何新平台。

✓

3 个新后端路由 + SSE 进度推送

GET /platforms/import-supported 列出支持 cookie 导入的平台;POST /importAccount 启动任务返回 task_id;GET /importAccount/stream?task_id=... SSE 推 4 步进度。cookie 失效自动报错并清理,不会留"百家号用户N"垃圾账号。

✓

4 步进度 UI 交互

步骤指示器 34px 圆形 SVG,状态图标动态切换(wait/process/finish/error);渐变进度条 (品牌色 → 浅色),失败时变红;步骤之间虚线连接;完成态账号预览卡片(头像 + 昵称 + 账号 ID)滑入动画 fade-up。失败时关闭按钮变红。
