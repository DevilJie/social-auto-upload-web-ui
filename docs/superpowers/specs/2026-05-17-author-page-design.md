## 三个版本设计方案

### 版本 A: 单页长卷 (Storytelling)
- **特点:** 自上而下滚动，类似故事线，沉浸感强
- **布局:**
  - Hero区：大号渐变标题 + 头像 + 标志性标语
  - 时间线/经历流：展示技术成长轨迹（3个阶段）
  - 技能标签云：横向排列技术栈
  - 项目展示：两个大卡片（博客、AI工厂）
  - 联系方式：社交链接 + 微信二维码
- **风格:** 深色背景 + 紫蓝渐变 + 时间线发光效果
- **适合:** 喜欢叙事、强调成长历程的用户

### 版本 B: 多模块网格 (Grid Mosaic)
- **特点:** 左右分栏，信息块分区拼贴
- **布局:**
  - 左侧：头像 + 名字标签 + 数据统计（10+年经验、50+项目）+ 技术栈标签 + 微信二维码
  - 右侧：个人简介 + 开源项目卡片 + 社交网络
- **风格:** 玻璃态卡片 + hover发光效果
- **适合:** 信息量大、需要清晰分类展示

### 版本 C: 中心辐射 (Cosmic)
- **特点:** 头像居中 + 周围环绕，酷炫感强
- **布局:**
  - 顶部提示条："正在播放: 程序员老蔡的故事"
  - 中心区：头像 + 双层环绕轨道动画
  - 周围：三个项目卡片环绕
  - 底部：关于我 + 技能标签 + 联系区
- **风格:** 星空背景 + 粒子动画 + 轨道旋转效果
- **适合:** 视觉效果优先、强调个人品牌
